# AnAuUno

AnAuUno is a unofficial Rust crate to interact with Android Auto–compatible devices.  
**Make Sure you have read the [State of Development](#state-of-development) section below.**

## State of Development

This crate is still in a very early stage of development, so don’t use it in production, as there are still bugs and 
breaking changes planned.
